### Install Python Modules
$ pipenv install flask flask-sqlalchemy flask-marshmallow marshmallow-sqlalchemy

$ pipenv install pylint-flask 

### Activate venv
$ pipenv shell

### Create DB by writing those on command
$ python
>> from app import db

>> db.create_all()

>> exit()

### Run Server
python app.py
